# afisha-telegram-bot

Берет информацию о событиях твоего города с сайта https://kassir.ru/ и отпровляет её. 


## Использование

Установите библиотеки с помощью команды:
```sh
$ pip install requests
$ pip install BeautifulSoup
$ pip install pyTelegramBotAPI
```
